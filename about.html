Online Library System

About

This is a mock online library system developed for the Web Development II Assignment by Keith Chambers

The system has the following functionality:

  • User can log in or register a new account
  • If the user is not logged in they will be refused access to the main page
  • Logged in users can search for books by auther and/or title
  • Search results will be limited to 5 per page and contain all stored information about the book
  • Users will be able to see in the search results what whether a book has been reserved or not. If a book is unreserved the user may reserve it him/herself if they have not exceeded their borrow limit
  • A logged in user may also see all the books they currently have reserved on the right side of the main page
